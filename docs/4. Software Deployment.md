# Software Deployment

## Deployment Unit

Rust service packaged by Cargo/Docker and deployed behind the gateway. Requires PostgreSQL, RabbitMQ for provisioning, and Midtrans environment variables when payment gateway is enabled.

## Local Stack

Run the full system from `bidmart-infrastructure` with Docker Compose. Service-specific test commands are still run from each repository for faster feedback.

## Configuration

Configuration comes from environment variables, compose files, and CI/CD secrets. Secrets must not be committed to the repository.

## Verification

Before promotion, run the module test suite and the infrastructure smoke/contract gates that exercise gateway-to-service wiring.

## Rollback

Rollback is done by redeploying the previous image/build or reverting the service commit, then rerunning smoke tests to ensure gateway wiring remains healthy.
