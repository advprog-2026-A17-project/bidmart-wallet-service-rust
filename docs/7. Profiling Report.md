# Profiling Report (Latency & Coverage)

Wallet is I/O-bound (Postgres, Midtrans HTTP, Rabbit). Evidence focuses on **llvm-cov** and **hold-path latency** under integration tests.

## Coverage

```bash
cargo llvm-cov --workspace --all-features --html
# CI uses --fail-under-lines 90
```

Artifact: CI `coverage-output.log`.
