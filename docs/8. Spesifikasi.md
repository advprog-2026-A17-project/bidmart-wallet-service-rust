# Spesifikasi — Modul 4 (Dompet dan Manajemen Saldo)

## Tanggung jawab

Ledger saldo aktif/hold, top-up & withdraw, hold/release/convert untuk lelang, provisioning otomatis.

## Fitur

- Wallet per user, riwayat transaksi
- Hold, release, convert (kontrak lelang)
- Top-up & withdrawal (Midtrans sandbox)
- Provisioning consumer dari registrasi auth
- Reconciliation hold kedaluwarsa (worker)
- gRPC untuk pemanggil internal
- Proteksi endpoint finansial dengan internal token

## Integrasi

| Modul | Arah |
|-------|------|
| 1 Auth | Event provisioning |
| 3 Lelang | HTTP hold/release/convert |
| 5 Order | Payout/refund seller (HTTP client) |
